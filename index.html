This page replicates a project for the Code Fellows 201 class that I TAd for two months. The top form adds a row to the table dynamically via JS. The second table adds a row to the table dynamically via jQuery, and sends that data as JSON to a firebase database file. Both tables render the data entered from the form.

The project context states that a small business owner wants to expand and wants each store to update their data so the owner can better plan central purchasing. Each store needs to report their minimum number of customers, the maximum number of customers and the average hourly purchases. The owner wants the script to return how many pounds of coffee are required for each store every day.

The "delete" button removes the respective row from the table, then deletes that data from the data storage.
